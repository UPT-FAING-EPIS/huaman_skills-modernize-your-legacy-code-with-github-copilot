# Test Plan - Legacy COBOL Account System

Este plan de pruebas valida la logica de negocio actual implementada en los programas COBOL (`MainProgram`, `Operations`, `DataProgram`) para su revision con stakeholders y posterior traduccion a pruebas unitarias e integracion en Node.js.

## Notas para validacion con stakeholders

- Este plan refleja el comportamiento actual del sistema COBOL, incluyendo ausencias de validacion (por ejemplo, montos cero).
- No se contempla persistencia a disco o base de datos: el saldo se mantiene en memoria durante la ejecucion.
- Los campos `Actual Result`, `Status (Pass/Fail)` y `Comments` se completan durante la ejecucion de pruebas.
